# How to Run

## For Backend

Install project dependencies:
```bash
npm install
```
Run the project:
```bash
npm start
```
Navigate to your browser and go to the specified port in `index.js` (default: 8080).
```
http://localhost:8080
```


## For Frontend

Install project dependencies:
```bash
npm install
```
Run the project:
```bash
npm run dev
```
Navigate to your browser and go to the specified port in `vite.config.js` (default: 5173).
```
http://localhost:5173
```
